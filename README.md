Command line to store environment variables in yaml file

### Install
```
npm i -g env2yaml
```

### Usage

```
$ env2yaml [VARIABLES]
```

### Arguments

```
VARIABLES Comma-separated list of variables to output
```

### Options

```
-h, --help show CLI help
-o, --output=output [default: env.yaml] Output file
-v, --version show CLI version
```
